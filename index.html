Please use the search function below to find information about current (and past) bills in the Utah State Legislature. If you know the "Bill ID"(i.e. HB 363), you can use the Search by Bill ID function, enter the Bill ID as well as the Session year, and find detailed information pertaining to the status of that bill. Otherwise, you can use the Keyword Search to learn more about bills in both the Utah House and Senate. Additionally, for a full list of bills (from 2011-2013), leave the search field blank and hit the Search button. *Remember to hit the Search button, otherwise the information you are looking for will not load.
